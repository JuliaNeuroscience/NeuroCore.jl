Units

Certain properties return measures with a Unitful unit type. These units all have default values but can be changed for a particular value type via method overloading.

NeuroCore.second_type — Function
second_type(x)

Returns the type used for seconds given x.

Examples

julia> using NeuroCore

julia> second_type(Any)
Unitful.Quantity{Float64,𝐓,Unitful.FreeUnits{(s,),𝐓,nothing}}
source
